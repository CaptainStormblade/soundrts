SoundRTS is a real-time strategy audio game.

The license for the source code is a BSD 3-clause license (LICENSE.txt).

Running the game from the source (with soundrts.bat for the client and run-server.bat for the standalone server) might require:
* Python 2.5
* Pygame 1.7.1
* win32com (if you use SAPI 5)

Building the binaries (with build-all.bat) might require:
* py2exe (Windows binary)

Official SoundRTS web site: http://jlpo.free.fr/soundrts
